# Assets

Coloque aqui os ícones e imagens do app:

- **icon.png** - 1024x1024px - Ícone principal do app
- **splash.png** - 1284x2778px - Tela de splash
- **adaptive-icon.png** - 1024x1024px - Ícone adaptativo Android
- **favicon.png** - 48x48px - Favicon para web

## Gerando Assets

Você pode usar ferramentas online ou criar manualmente:

1. **Icon Generator**: https://www.appicon.co/
2. **Splash Generator**: https://www.appsplashscreen.com/
3. **Ou usar Expo**: `npx expo-optimize` (comprime imagens)

Por enquanto, o app usará placeholders padrão do Expo.
